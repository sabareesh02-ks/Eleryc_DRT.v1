üõ°Ô∏è TROUBLESHOOTING (PERMANENT SOLUTIONS):
================================================================================

Problem: "Failed to fetch"
Solution: Start Flask server (START_AUTOMATED_SERVER.bat)

Problem: "No CSV files found"
Solution: Check folder structure in separated_by_current/

Problem: Experiment not in list
Solution: Verify folder name matches exactly, refresh browser

Problem: Server won't start (port busy)
Solution: 
   1. Find process: netstat -ano | findstr :5000
   2. Kill it: taskkill /F /PID [process_id]
   3. Restart: START_AUTOMATED_SERVER.bat

Problem: Wrong data in plot
Solution: Verify CSV format (DRT has 2 header rows)

Problem: Plots don't appear
Solution: Hard refresh browser (Ctrl+F5)
